在IPv6 Only VPS上使用Warp

2022-09-13 223 字
Cloudflare Warp VPS

前言

鉴于Privex家的VPS一般只提供IPv6地址

而有些网站不支持IPv6访问

因此选择CloudFlare Warp的第三方客户端WgCF为VPS增添IPv4支持

WgCF

  • 本部分包含安装与配置

安装

WgCF

可在GitHub上找到WgCF的releases

wget https://github.com/ViRb3/wgcf/releases/download/v2.2.15/wgcf_2.2.15_linux_amd64

mv wgcf_2.2.15_linux_amd64 wgcf

chmod +x wgcf

WireGuard

  • 安装 apt install wireguard-tools openresolv -y

配置

WgCF

下载完成后，可以使用./wgcf register注册Warp账号

然后使用./wgcf generate生成配置文件 wgcf-profile.conf

使用nano编辑 wgcf-profile.conf 中的[Peer]部分

[Peer]
PublicKey = "Your Key"

AllowedIPs = 0.0.0.0/0

Endpoint = [2606:4700:d0::a29f:c001]:2408

WireGuard

将 wgcf-profile.conf 移动至WireGuard目录

sudo cp wgcf-profile.conf /etc/wireguard/wgcf.conf

设置开机自启 sudo systemctl enable wg-quick@wgcf.service

结尾

至此，你已经成功地设置了IPv4上的Warp

  • 关于MTU的优化，可以根据VPS的实际情况调整
